Set up Kali Linux

Quick starter guide to installing and preparing Kali.

Install Kali

Download the ISO from the official Kali website and create a bootable USB or VM.

Update packages

sudo apt update && sudo apt full-upgrade -y

Keep the system and tools up to date.

Enable root or sudo

sudo passwd root
sudo usermod -aG sudo

Set up a privileged account for lab testing.
