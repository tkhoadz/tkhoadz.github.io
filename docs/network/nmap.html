Nmap Scan Examples

Basic network discovery and port scanning commands for Kali Linux.

Quick network discovery

sudo nmap -sn 192.168.1.0/24

Performs a ping scan to list live hosts on the subnet.

Service and version scan

sudo nmap -sV -p 1-1000 192.168.1.10

Scans the first 1000 ports and attempts to identify services.
