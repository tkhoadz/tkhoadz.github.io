Wireless Defense Awareness

Recognize wireless attack vectors and learn how defenders can detect them.

Scan visible networks

sudo iwlist wlan0 scan | grep -E 'ESSID|Signal'

Review nearby SSIDs and signal strength to identify unexpected or suspicious wireless access points.

Check interface mode

sudo iw dev wlan0 info

Validate that your wireless adapter is in managed mode, not monitor mode, unless you are using it for authorized analysis.

Spot rogue devices

Defenders should be able to distinguish between trusted access points and unknown devices broadcasting Wi-Fi.
