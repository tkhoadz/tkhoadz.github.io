Nikto Web Scanner Awareness

Scan your test web servers for common misconfigurations and vulnerabilities.

Basic scan

nikto -h http://192.168.1.100

Run default scan on lab web server, checking for outdated software, misconfigs.

Save output

nikto -h http://192.168.1.100 -o nikto-scan.html -Format html

Generate HTML/CSV reports for review. Pipe to tools like grep for specific issues.

SSL and plugins

nikto -h https://192.168.1.100 -ssl -Plugins "headers,software"

Target HTTPS, limit plugins to focus on headers/server info. Test evasion with -Tuning.
