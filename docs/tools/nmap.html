Nmap Reference

Common Nmap command examples for Kali Linux.

Top ports scan

sudo nmap -T4 -A -p- 192.168.1.10

Scans all ports with aggressive detection and service enumeration.

Save output to file

sudo nmap -oA output 192.168.1.10

Saves normal, XML, and grepable output files for later analysis.
